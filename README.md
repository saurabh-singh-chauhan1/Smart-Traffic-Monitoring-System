# Smart Traffic Monitoring System: 
A Smart Traffic Monitoring System is an intelligent solution that uses computer vision and deep learning to automatically detect, track, and analyze vehicles in real-time from images or video streams. It helps in improving traffic management, reducing congestion, and enhancing road safety.

# Features:
* Real-time vehicle detection (YOLOv8)
* Multi-object tracking with unique IDs (DeepSORT)
* Vehicle counting
* Works on images and videos
* High accuracy with optimized models
* Supports smart traffic analysis

# Tech Stack:
* Language: Python
* Model: YOLOv8 (Ultralytics)
* Tracking: DeepSORT
* Libraries: OpenCV, NumPy, Matplotlib
* Platform: Kaggle and Local System
